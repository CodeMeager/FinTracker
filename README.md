# Мини-сайт "Трекер финансов"

Веб-приложение на **React + TypeScript** для контроля финансов.

### Как открыть?

1. Убедитесь, что у вас установлен **Node.js**.
2. В корне проекта выполните:

```bash
npm install
npm run dev
```

3. Откройте указанный адрес в браузере (Терминал напишет точный URL в консоль).

# !Внимание! Трекер обнуляется при перезагрузке страницы
